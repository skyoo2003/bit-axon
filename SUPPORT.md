# Support

## How to Get Help

- **GitHub Discussions**: Ask questions and share ideas at [github.com/skyoo2003/bit-axon/discussions](https://github.com/skyoo2003/bit-axon/discussions)
- **Bug Reports**: If you found a bug, please [open an issue](https://github.com/skyoo2003/bit-axon/issues/new/choose) with a clear description and reproduction steps.
- **Feature Requests**: Share your ideas in [Discussions](https://github.com/skyoo2003/bit-axon/discussions) first, then open an issue if approved by maintainers.

## Before Asking

1. Check the [README](README.md) and [documentation](docs/) for existing answers
2. Search [existing issues](https://github.com/skyoo2003/bit-axon/issues) and discussions
3. Make sure you're using the latest version: `pip install --upgrade bit-axon`

## Reporting Issues

When reporting a bug, please include:
- Bit-Axon version (`python -c "import bit_axon; print(bit_axon.__version__)"`)
- Python version
- macOS version and Apple Silicon model (M1/M2/M3/M4)
- Minimal reproduction code
- Expected vs actual behavior
