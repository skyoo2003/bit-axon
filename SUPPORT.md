# Support

## How to Get Help

- **Documentation**: Full guides at [skyoo2003.github.io/bit-axon/](https://skyoo2003.github.io/bit-axon/)
- **FAQ**: Common questions answered in [FAQ](https://skyoo2003.github.io/bit-axon/faq/)
- **GitHub Discussions**: Ask questions and share ideas at [github.com/skyoo2003/bit-axon/discussions](https://github.com/skyoo2003/bit-axon/discussions)
- **Bug Reports**: [Open an issue](https://github.com/skyoo2003/bit-axon/issues/new/choose) with a clear description and reproduction steps
- **Feature Requests**: Share your ideas in [Discussions](https://github.com/skyoo2003/bit-axon/discussions) first

## Before Asking

1. Check the [README](README.md) and [documentation](https://skyoo2003.github.io/bit-axon/) for existing answers
2. Search [existing issues](https://github.com/skyoo2003/bit-axon/issues) and [Discussions](https://github.com/skyoo2003/bit-axon/discussions)
3. Make sure you're using the latest version: `pip install --upgrade bit-axon`

## Reporting Issues

When reporting a bug, please include:
- Bit-Axon version (`python -c "import bit_axon; print(bit_axon.__version__)"`)
- Python version (`python --version`)
- macOS version and Apple Silicon model (M1/M2/M3/M4)
- Available memory (`sysctl -n hw.memsize`)
- Minimal reproduction code
- Expected vs actual behavior
- Relevant log output or stack traces

## Response Times

The maintainer is a solo developer working on this project in spare time. Responses may take several days depending on volume and complexity.
