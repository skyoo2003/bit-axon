# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅        |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public issues.**

Instead, please report them via [GitHub Private Vulnerability Reporting](https://github.com/skyoo2003/bit-axon/security/advisories/new).

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 7 days
- **Fix timeline**: Communicated within 14 days

## Security Measures

This project uses:
- [Dependabot](https://docs.github.com/en/code-security/dependabot) for automated dependency updates
- [Secret scanning](https://docs.github.com/en/code-security/secret-scanning) and push protection
